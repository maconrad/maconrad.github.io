Pelican Blogging

Posted on So 24 Oktober 2021 in Random

Installing and using pelican for blogging

Easy, nice and minimalistic approach for blog based on static content and capable of RST as well as markdown including support for pygments for code highlighting.

Can it get any more perfect for a simple blog like mine? 

# pelican specific commands
pip install pelican[Markdown]
# Starting a new project
pelican-quickstart
# Create content manually
pelican content 
# Create content manually
make html
# Serve content on server locally
make serve
# Watch files
make devserver

Thanks @pelican-devs. Awesome job.

Following the push flow for git

cd output
make clean
make html
git init
git add .
git commit -m "msg"
git branch -M main
git remote add origin https://github.com/maconrad/pelicanblog.git
git push -u origin main
