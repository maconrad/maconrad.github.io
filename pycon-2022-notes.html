OpenStreetmap with Python

Posgres / OSM / python

OSM

OSM has 3 Data Types - Nodes - Waste Baskets - Toilets - ... - Way (connection of nodes) - Open way (line) - Closed way (starts and stops) - Roundabout / parks (areas) - Relations (group nodes, ways and other relations) - Lakes with islands - university campuses - bus routes - Catogorize & search with tagging - Key&values - Examples (amenity e.g. cinema, post_box, townhall, waste_basket) - Examples (landuse e.g. commercial, industrial, residental, farmland, forrest)

OSM data has PBF binary format (fastest to read and write).

OSM and Postgres

Geospatial Potgres - PostGIS Plugin (implements standards = basic geographic datatypes) - Opengeospecial consortium (geospaical analysis) - Simple features for SQL (SFSQL) - poligon (find all points in that poligon)

Geometry Objects - Poligon / POint / Linestring - Geometry collection (multipoint)

PostGIS Introduction URL for how to write spacial queries with PostGIS

Tools to get Data OSM Data into DB - osmium (extract, merge commands) - E.g. extract berlin - osm2pgsql (not very descriptive table names) - Lua scripts to modify

OSM and python

Use CLI like click, rich for fast development cycle. Plotly to write chart in html, richt to write nice table. Poetry to deliver package
