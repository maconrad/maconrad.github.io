Overview

  • Starting an App
    • React Native (access to native code, swift/java)
    • Expo (simplifies getting started, own sdk, client app, run expo project in borwser for testing, react native api + expo sdk only, eject if want to include native code)
    • IGNITE CLI (https://github.com/infinitered/ignite)
  • Styling, recommended to use a UI Toolkit, Options:
    • NativeBase
    • ReactNativeElements
  • Routing
    • React Navigation (move from screen to screen)
      • Keeps recored of navigation history
      • Examples: Stack (Deck of cards), TabNavigator (commonly at top or bottom), Drawer (Pulls out from side of screen)
    • React Native Router FLux
  • DataStorage
    • AsyncStorage: Key-value storage, unencrypted, asynchronous, persistent on device
    • DB Backend: REST

Mini Learnings

  • NPX = Tool to execute NPM packages from NPM Registry (not installed)
  • NPM = Manages packages but does NOT execute them
  • Hooks = Functions that encapsulate stateful behaviour and side effects (distributed state)
    • Common ones: useState, useEffect, useContext
  • Metro = Returns single JS File that includes all code and dependencies

Installation

  • Tools Below + Android SDK + XCode (on MAC):
    • ~./zshrc: For Android SDK Path Vars
  • XCode
    • Settings > Locations > Select XCode Version otherwise pod installbelow will fail
