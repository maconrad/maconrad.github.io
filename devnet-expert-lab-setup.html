Exam Topics

The DEVNet Lab exam consists of the following high level topics: - Design - IaC - Network Programmability (REST APIs) - Network Automation - Containers - Security

The DEVNet Lab exam consists of the following high level technologies: - Git - Python - Ansible - Terraform - Docker - NSO - pyATS - Telemetry - Kubernetes

In addition the following should be familiar: - TIG Stack: Telegraf (1.19), InfluxDB 1.8, Grafana 8.1 - GitLab & Gitlab Runner 14 - HashiCorp Vault 1.8 - K8s 1.23

Following software will be used: - Nexus 9k 9.3(9) - ACI Simulator 5.2 - Cat8000V: 17.5 - IOSv: 15.9 - IOSvL2: 15.2
